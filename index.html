🔌  PIN CONNECTIONS — ARDUINO UNO
Rainfall AO→ A0Analog rainfall intensity
Soil Moisture AO→ A1Analog soil saturation
Water Level S→ A2Analog water depth
DHT11 DATA→ D2Temp + Humidity (+ 10kΩ to 5V)
BMP180 SDA→ A4I2C data
BMP180 SCL→ A5I2C clock
All VCC pins→ 5VPower
All GND pins→ GNDGround
USB cable→ PCSerial data + power
⚠️ DHT11 requires a 10kΩ pull-up resistor
between the 5V pin and the DATA pin (D2).
Without it, temperature readings will fail.
📦  COMPONENTS LIST
1× Arduino UNO
1× Rainfall Sensor Module (YL-83)
1× Capacitive Soil Moisture Sensor
1× Analog Water Level Sensor
1× DHT11 Temperature & Humidity
1× BMP180 Barometric Pressure (I2C)
1× 10kΩ resistor
1× Breadboard + jumper wires
1× USB A-to-B cable
1× PC / Laptop (runs serial_bridge.py)
🔄  DATA FLOW
Sensors → read every 30s → Arduino → JSON over USB Serial → serial_bridge.py → HTTP POST → Flask API (Render) → ML ensemble → Prediction response + Twilio SMS (if risk ≥ 50%)
☁️  BACKEND — RENDER.COM (FREE)
1
Push to GitHub
Create a repo on github.com and push the entire FloodSense folder.
Make sure the models/ folder with .pkl files is included.
2
Create a Web Service on Render.com
New + → Web Service → connect GitHub repo
Build command: pip install -r requirements.txt
Start command: gunicorn app:app --workers 2 --bind 0.0.0.0:$PORT
Plan: Free
3
Add Environment Variables
TWILIO_ACCOUNT_SID  = ACxxxx...
TWILIO_AUTH_TOKEN   = xxxx...
TWILIO_FROM_NUMBER  = +1XXXXXXXXXX
ALERT_PHONE_NUMBERS = +919XXXXXXXXX
4
Connect this Dashboard
After deploy, your API URL will be:
https://floodsense-api.onrender.com
Click ⚙️ Settings above and paste your URL there.
🌐  FRONTEND — GITHUB PAGES (FREE, THIS FILE)
1
Create a separate GitHub repo for the frontend
Name it: floodsense-dashboard
Upload: index.html, 404.html, _config.yml
2
Enable GitHub Pages
Repo → Settings → Pages
Source: Deploy from a branch → main → / (root)
Click Save. Live in ~60 seconds at:
https://YOUR_USERNAME.github.io/floodsense-dashboard
3
Set your API URL in Settings
Open the live GitHub Pages URL in any browser.
Click ⚙️ → paste your Render URL → Save.
The dashboard will immediately connect to the live ML API.
